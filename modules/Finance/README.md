# KAVAC | Modulo de Finanzas

Módulo que permite gestionar Finanzas de la organización bajo las siguientes funcionalidades

* Configuración
  * Formatos de códigos
  * Registros comunes
    * Bancos
    * Agencias bancarias
    * Tipos de cuenta
    * Cuentas bancarias
    * Formas de pago
* Gestión de pagos
  * Órdenes de pago
  * Emisiones de pago
* Banco
  * Movimientos
